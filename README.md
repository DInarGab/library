# Библиотечный Telegram-bot

Возможности -
1) Сбор индексация книг в библиотеке
2) Выдача книг
3) Уведомление о необходимости сдачи книги
4) Возможность предложить книгу на добавление в библиотеку. Поддерживаемые сервисы для парсинга - Лабиринт, Читай город.

# Команды

Для всех пользователей
/start — начало работы с ботом, регистрация/авторизация
/list_books — просмотр списка доступных книг 
/suggest_book — предложить новую книгу для добавления в библиотеку (можно ввести данные вручную или отправить ссылку на книжный магазин).
/list_lendings — просмотр книг, которые у вас на руках
/list_suggestions — просмотр статуса предложенных пользователем книг (для админа показываются все)

Для администраторов
/add_book — добавить новую книгу в библиотеку

И остальные команды дают дополнительный функционал при вызове
/list_books - дает возможность выдать книгу пользователю
/list_suggestions - дает возможность одобрить или отклонить предложение
/list_lendings - показывает все выдачи пользователей и позволяет принять книгу обратно
